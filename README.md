# ai-code-auditor

AST-based secure code auditing linter and LLM-powered helper to find vulnerabilities and code smells.

A premium, high-quality repository maintained by **Novalabs-in**.

## 🚀 Key Features
- Stateful, robust, and clean Python implementation.
- Standard configuration settings with modular codebase patterns.
- Extensively optimized for performance, scalability, and ease of deployment.

## 🛠️ Tech Stack & Dependencies
- **Core Language**: Python
- **Dependencies** (defined in `requirements.txt`):
```text
astroid>=2.15.0
pylint>=2.17.0
openai>=1.0.0
```

## ⚙️ Setup and Installation
1. Clone this repository under your workspace:
   ```bash
   git clone https://github.com/Novalabs-in/ai-code-auditor.git
   cd ai-code-auditor
   ```
2. Set up and activate a local virtual environment:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   ```
3. Install all dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Execute the application:
   ```bash
   python main.py
   ```
